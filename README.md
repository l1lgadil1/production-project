## BLOG project

First, run the development server:

```bash
npm install
```
or
```bash
yarn 
```

```bash
npm run start:dev
```
```bash
yarn run start:dev
```

## How to launch tests:

Run the following command:
```bash
npm run test:unit 
```
To run specific test:
```bash
npm run test:unit (testname) 
```

or

Run the following command:
```bash
yarn run test:unit 
```
To run specific test:
```bash
yarn run test:unit (testname) 
```
